# data-bank

Projeto em grupo desenvolvido em C#, Asp.net e SQL Server no processo seletivo que conseguiu nosso emprego na empresa Datacoper;<br/>
É um gerenciador bancário, onde é possível realizar diversos cadastros, como filiais, empresas, usuários, contas bancárias, etc;<br/>
Também é possível realizar transferências, estornos e pagamentos entre contas, e visualização de extrato;<br/>
Primeiro login de acesso apenas com a conta admin/admin;<br/>
